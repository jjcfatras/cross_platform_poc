# Setting up Dev Environment

### Run Following Commands from Project Root

1. `nvm use`
2. `corepack enable`
3. `yarn`

## Using Web

### Launch Dev Server

```sh
yarn nx run web2:dev
```

## Using Mobile

### Create Development Build

```sh
yarn nx run mobile:run-ios
```

### Startup Existing Build

```sh
yarn nx run mobile:start
```
